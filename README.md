# My dotfiles

This repo organized around [GNU Stow](https://www.gnu.org/software/stow/).

## Usage examples
On Linux laptop:
```sh
stow base hyprland-with-uwsm linux-laptop
```

On Linux desktop:
```sh
stow base hyprland-with-uwsm linux-desktop
```

On MacOS:
```sh
stow base macos
```
